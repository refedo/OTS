METHOD 1: VISUAL EDITOR (No coding needed!)
-------------------------------------------
1. Open this file in your browser:
   preview-delivery-note.html

2. Use the color pickers on the right side

3. Click "Apply Changes" to see results instantly

4. Copy the color values you like

5. Update styles.css with those values

6. Run: node test-delivery-note.js

7. Done!


METHOD 2: DIRECT EDITING (Full control!)
-----------------------------------------
1. Open this file:
   src/modules/reporting/templates/delivery-note/styles.css

2. Find the color you want to change (e.g., line 50)

3. Change the value:
   background: #E74C3C;  ‚Üí  background: #0071BC;

4. Save the file

5. Run: node test-delivery-note.js

6. Open the generated PDF

7. Repeat until perfect!
