## ğŸ¯ How to Create Your First Notification

### Method 1: Via Prisma Studio (Quick Test)

1. Open http://localhost:5555
2. Click "notifications" table
3. Click "Add record"
4. Fill in:
   - `userId`: Your user ID
   - `type`: SYSTEM
   - `title`: "Test Notification"
   - `message`: "Testing the notification system"
   - `isRead`: false
5. Save
6. Go to `/notifications` page
7. See your notification!

### Method 2: Via Code (Production)

```typescript
import NotificationService from '@/lib/services/notification.service';

await NotificationService.notifySystem({
  userId: 'your-user-id',
  title: 'Welcome!',
  message: 'Your notification system is active.',
});
```
